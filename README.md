# App Store スクリーンショット リサイザー

App Store申請用のスクリーンショットを簡単にリサイズできるWebアプリケーションです。

## 🌟 機能

- **画像アップロード**: ドラッグ&ドロップまたはクリックで画像を選択
- **プリセットサイズ**: iPhone/iPad Proの主要なスクリーンショットサイズを用意
- **カスタムサイズ**: 任意のサイズを指定可能
- **クロップモード**:
  - 横基準でクロップ: 幅を基準に拡大してクロップ
  - 縦基準でクロップ: 高さを基準に拡大してクロップ
- **ドラッグ調整**: クロップ領域をドラッグで調整可能
- **リアルタイムプレビュー**: 変更をすぐに確認
- **iPhone対応**: モバイルデバイスでも快適に使用可能

## 🚀 使い方

1. [こちらからアクセス](https://yourusername.github.io/pic-resizer/)
2. 画像をドラッグ&ドロップまたは選択
3. プリセットまたはカスタムサイズを選択
4. クロップモードを選択（横基準/縦基準）
5. ドラッグしてクロップ位置を調整
6. ダウンロードボタンで保存

## 📱 対応サイズ

- iPhone 6.7" (1290×2796)
- iPhone 6.5" (1242×2688)
- iPhone 5.5" (1242×2208)
- iPad Pro 12.9" (2048×2732)
- iPad Pro 11" (2048×2732)
- カスタムサイズ
